Tabs

The tabs mixin configures a tab navigation and a tab content area, respectively the first and second (and only) children of the element where the tabs() mixin is used.

Use Params Notes Examples

Tabs - Basic Use

HTML


<div class="tabs">
  <div>
    <span>Tab 1</span>
    <span>Tab 2</span>
    <span>Tab 3</span>
  </div>
  <div>
    <div>Tab 1 content.</div>
    <div>Tab 2 content.</div>
    <div>Tab 3 content.</div>
  </div>
</div>

Stylus


.tabs
  tabs()

jQuery


$('.tabs > div:first-child').on('click', function (e) {
  var index = $(e.target).index();
  $(this).next().children().removeClass('active').eq(index).addClass('active');
  $(this).children().removeClass('active').eq(index).addClass('active');
});
